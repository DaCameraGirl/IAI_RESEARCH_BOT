# 26016 Hymn Research - Italian

Status: active
Category: Other/Miscellaneous
Research type: Copyright Research
Expiration date: **2026-07-30**

## Goal

Find existing Italian translations of hymns from the RWS-provided hymn list
(`HymnResearch_English.zip`). Do not create, machine-translate, AI-translate,
or infer translations. Each submission must correspond to one Italian
translation of one hymn from one source.

## Required Submission Fields

- English hymn name for which the translation is being submitted
- Complete translated hymn text, including translated title
- Translator name, including birth/death date and place if verified
- Date translation was published, if known
- Where the translation was published/identified: publication title, date,
  place, page number, URL, or equivalent bibliographic details
- Do not guess missing translator, date, place, or publication details —
  leave out anything unverified.

## Attachment Requirement

Each submission needs a PDF attachment including one of:

- images from the hymnal/document (cover, copyright page, hymn text with
  musical accompaniment; translator-supporting images welcome);
- website screenshots directly from the submitted URL.

Submitter will be asked to highlight the key identifying info in the
attachment.

## Submission Rule

Each submission must cover a single Italian translation for a single hymn
from a single source. Do not bundle multiple hymns/translations/sources.

## Alternate Submission: Search Strategy Proof

If RWS requests proof a hymn was searched but nothing was found, a search
strategy submission is also in-scope:

- hymn searched
- databases searched
- search strings used
- results summary (number of results; what was/wasn't identified)

## Hard Filters

- Do not show self-made, machine-translated, or AI-translated hymns.
- Do not show paywalled/inaccessible sources unless Angela already has
  access and asks to use them.
- Do not duplicate prior submissions or the same source/translation.
- Prefer sources with visible Italian text, source title, date/publication
  context, and screenshots/PDF pages that can be highlighted.

## Scoring Criteria (best collections)

- total number of hymns with a submitted existing translation
- overall quality (all required info + sources documented)
- diversity (multiple religious denominations represented)
- search strategies submitted (esp. for languages/hymns with no results)
- rare hymns found
- physical hymn books searched

## First Search Lanes

- Italian hymnals across Catholic, Waldensian, Baptist, SDA, Methodist,
  Pentecostal denominations.
- National Libraries (Biblioteca Nazionale catalogs, etc.), denominational
  hymnal indexes (iOS/Android apps included), Internet Archive, Google
  Books, HathiTrust, YouTube.
- Search by full first line of the hymn in quotes, not just title; also try
  tune name / composer + keywords.
